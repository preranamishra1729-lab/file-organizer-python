# File Organizer Script

## What it does
Automatically sorts files into folders by type (Images, Documents, Audio, etc.)

## How to run
1. Install Python
2. Run: python file_organizer.py
3. Enter folder path when asked

## Requirements
- Python 3.x
- OS module (built-in)

## Sample Output
Files get sorted into: Images/, Documents/, Audio/, etc.
